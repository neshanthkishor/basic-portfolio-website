This project is a simple and responsive personal portfolio website developed using HTML and CSS. It is designed to showcase my profile, skills, projects, and contact information in a clean and user-friendly layout. The website includes sections such as Home, About, Skills, Projects, and Contact, making it easy for visitors to learn more about me and my work. This project helped me strengthen my understanding of HTML, CSS, responsive web design, and creating modern user interfaces
The portfolio follows a modern design with a responsive layout that adapts to different screen sizes. It uses a navigation bar for easy access to each section and includes interactive elements such as hover effects to enhance the user experience. The project demonstrates the fundamentals of front-end web development and responsive web design.
Developing this portfolio helped me improve my knowledge of HTML structure, CSS styling, page layout, and responsive design techniques. It also provided practical experience in creating an organized and visually appealing website that can be used to present personal information and projects professionally.
